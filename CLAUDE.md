# Project Context

A local demo that maps observable social cues (tone, expression, context, literal meaning) through a weighted Neo4j graph to likely social interpretations, summarized by Claude.

**Stack:** Neo4j · Python · TypeScript/Node.js (Express) · Cytoscape.js · Claude API (`@anthropic-ai/sdk`)

**Layout:**
- `graph_design/spec/graph_specification.md` — source of truth for the graph schema and sample data
- `graph_design/pipeline/` — Python scripts to generate Cypher (via Claude), load into Neo4j, and export to Cytoscape JSON
- `graph_design/cypher/` — timestamped Cypher and schema docs generated by the pipeline
- `app/server/src/` — Express API: Neo4j queries, Claude interpretation, routing
- `app/web/` — Cytoscape.js UI

**Run the pipeline** (rebuilds Neo4j from the spec):
```powershell
.\graph_design\pipeline\run_graph_pipeline.ps1
```

**Run the app:**
```bash
npm run dev  # http://localhost:3000
```

# Code Change Philosophy

- Fix the specific problem with the minimum change. Never rewrite a working script without explicit approval.
- If a change makes a file significantly longer, reconsider the approach.
- If a fix introduces new errors, stop and offer to revert rather than continuing to patch.
- Do not expand prompts or payloads unless explicitly asked.
- Make one small change at a time, verify it works, then continue.
